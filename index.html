Search Engine

A rudimentary search engine using a graph implementation of Larry Page's PageRank algorithm to sort search results.
Source not posted in accordance with UW academic policy.

Java PageRank Coursework

Concurrent Task Scheduler

A task scheduling library. The task scheduler runs tasks concurrently on a threadpool when possible. Tasks may be given dependencies, only allowing them to run when another task is finished. Tasks are stored in a lock-free work-stealing queue.
This project was for learning purposes, not for use in production.

C++ Concurrency Personal

Maze Creator + Solver

Implements Kruskal's Minimum Spanning Tree algorithm to generate a solveable maze, and Djikstra's shortest path algorithm to find the shortest solution to the maze.
Source not posted in accordance with UW accademic policy.

Java Shortest Path Coursework

Sudoku Solver

Implements recursive backtracking to solve a sudoku board. Game board is created using a text input file, providing pre filled values.

C++ Recursive Backtracking Personal

Fuzzy String Matcher

A fuzzy string searcher that allows the user to type in a search parameter, and returns approximate string matches. Uses Peter Norvig's spell checking algorithm, along with a Longest Common Substring algorithm to calculate edit distance.

C++ Pattern Matching Personal

Midi Player

Reads input from a MIDI keyboard (or any MIDI device) and executes the appropriate action for the read MIDI input. For example, a MIDI Note event outputs sound, and a Controller event changes the instrument.

C# Device Read/Write Personal
